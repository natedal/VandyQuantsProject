# Option Pricing Web App

This  application allows users to calculate option prices using the Black-Scholes model and visualize the
lognormal distribution of the stock price at expiration.



## Features

- Calculate call and put option prices using the Black-Scholes formula.
- Input parameters: Stock Price (S), Strike Price (K), Time to Expiration (T), Risk-free Rate (r), Volatility (Ïƒ).
- Visualize the lognormal distribution of the stock price at expiration.
